
## CLI工具使用
快速生成node-packge的脚手架

## useage
```bash
# 安装package
npm i vonxq-create-node-package -g
# 新建node-package目录
create-package dva package-name 
create-package basic package-name 
create-package babel package-name 
create-package default package-name 
# 测试packge
cd package-name
npm link
sayHi
```
options:
--install是否安装依赖
